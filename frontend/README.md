# Team Task Tracker

A full-stack task management application where users can create teams, assign tasks, and manage team work efficiently.

## Tech Stack

Frontend:
- React + Vite
- CSS

Backend:
- Flask
- SQLAlchemy
- SQLite
- bcrypt password hashing

---

# Local Setup

## Prerequisites

Make sure you have installed:

- Node.js
- Python 3.10+
- Git

---

## Backend Setup

Open a terminal:
cd backend
pip install -r requirements.txt
python app.py


#Frontend Setup
cd frontend
npm install
npm run dev
