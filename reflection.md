# Reflection

## What tier did you complete?

I completed the Easy and most of the Medium requirements, including user authentication, personal and team task management, team creation, task assignment, status updates, and connecting a React frontend with a Flask backend using SQLite. I also added features such as an admin dashboard and a profile page. However, I wasn't able to complete some of the more advanced requirements, such as displaying task counts per team and implementing a view that shows all tasks under a team, including its sub-teams. I also wasn't able to spend as much time refining the UI , as most of my time went into getting the core functions working.

## What was the hardest part?

The hardest part was learning the technologies while building the project. I had not worked with Flask, React, or full-stack development before, so I was learning as I went. Understanding how the React frontend communicates with the Flask backend and how data is stored and retrieved from SQLite took time. Since many of these concepts were completely new to me, I wasn't able to finish every feature within the given time, but the project gave me a much better understanding of how a full-stack application is built.

## If I could do it again, what would I do differently?

If I were to do this project again, I would start from the beginning with a better understanding of how the frontend, backend, and database connect with each other. Since I was learning Flask, React, and full-stack development while building the project, I spent a lot of time figuring out concepts as I went. Now that I understand the overall structure much better, I would organize the project more effectively, build features in a logical order, and avoid some of the debugging and rework that I encountered. I would also spend more time with the UI.

## One thing I'm proud of

I'm proud that I built a full-stack application from scratch using technologies that were mostly new to me. The flow, task management features, Through this project, I also learned how to use bcrypt for secure password hashing, how frontend and backend integration works, and also to writing integration and unit tests. Even though I couldn't complete every feature, I'm proud of how much I learned and how much I was able to build in a short time.

## What I want to learn next

This project made me realize that I want to strengthen my full-stack development skills. I want to learn more about Docker,  deployment of backend. I also want to be able to build complete applications on my own without relying heavily on AI tools.
