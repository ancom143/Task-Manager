# Reflection

## What tier did you complete?

I completed the Easy and Medium requirements successfully, including user authentication, personal and team task management, team creation, task assignment, status updates, and a responsive React frontend connected to a Flask backend using SQLite. I also implemented additional features like an admin dashboard, profile page, and improved UI styling. I attempted some advanced features such as nested teams and deployment, but I wasn't able to polish every part within the given time.

## What was the hardest part?

The hardest part was connecting the React frontend with the Flask backend while keeping the application state updated correctly. Since this was one of my first full-stack projects, understanding how data flowed between the frontend, backend, and SQLite database took time. Debugging API calls, fixing CORS issues, handling routing, and making sure tasks refreshed correctly after every action were the biggest challenges. Deployment was also difficult because it introduced another set of configuration issues that I hadn't dealt with before.

## If I could do it again, what would I do differently?

I would spend more time planning the project structure before writing code. I often implemented features first and cleaned them up later, which led to extra debugging. I would also commit my changes to Git more frequently with smaller commits and write simple tests while building each feature instead of leaving testing until the end.

## One thing I'm proud of

I'm proud that I built a complete full-stack application from scratch using technologies that were mostly new to me. The authentication flow, task management features, and separation between personal and team tasks work well together. I also improved the user interface to make the application more organized and easier to use.

## What I want to learn next

This project showed me that I need a stronger understanding of full-stack development. I want to learn more about Docker, automated testing, CI/CD pipelines, authentication best practices, and deployment. I also want to improve my knowledge of React state management and backend architecture so I can build larger applications with cleaner and more maintainable code.
